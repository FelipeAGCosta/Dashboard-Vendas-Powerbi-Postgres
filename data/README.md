# Dataset (Kaggle)

O arquivo completo do dataset **não é versionado** neste repositório (tamanho/licença).

## Como baixar
1. Baixe o dataset no Kaggle
2. Salve o CSV principal como:
   - `data/vendas_kaggle.csv`

## Observação
Se necessário, este repositório pode conter um arquivo de amostra com poucas linhas:
- `data/amostra_vendas.csv`
