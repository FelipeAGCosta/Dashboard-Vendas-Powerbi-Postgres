# Dashboard de Vendas (Kaggle → PostgreSQL → Power BI)

Pipeline simples de dados usando dataset real do Kaggle, carregado em PostgreSQL e consumido no Power BI para criar um dashboard interativo de vendas.

## Arquitetura
Kaggle (CSV) → PostgreSQL → Power BI

## Tecnologias
- PostgreSQL
- Power BI Desktop
- (Opcional) Python: pandas + SQLAlchemy/psycopg2

## KPIs (planejado)
- Faturamento total
- Pedidos (distintos)
- Clientes únicos
- Ticket médio
- Vendas por período e categoria
- Top produtos por faturamento
